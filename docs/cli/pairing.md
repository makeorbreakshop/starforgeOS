# `starforge pairing`

Approve or inspect DM pairing requests (for channels that support pairing).

Related:

- Pairing flow: [Pairing](/channels/pairing)

## Commands

```bash
starforge pairing list whatsapp
starforge pairing approve whatsapp <code> --notify
```
