# `starforge tui`

Open the terminal UI connected to the Gateway.

Related:

- TUI guide: [TUI](/web/tui)

## Examples

```bash
starforge tui
starforge tui --url ws://127.0.0.1:18789 --token <token>
starforge tui --session main --deliver
```
