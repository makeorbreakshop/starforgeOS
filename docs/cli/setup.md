# `starforge setup`

Initialize `~/.starforgeos/starforgeos.json` and the agent workspace.

Related:

- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
starforge setup
starforge setup --workspace ~/.starforgeos/workspace
```

To run the wizard via setup:

```bash
starforge setup --wizard
```
