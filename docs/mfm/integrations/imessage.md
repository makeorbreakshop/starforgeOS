# iMessage

Relay via BlueBubbles on Mac Mini.

## Setup

- **Server**: BlueBubbles running on Mac Mini (Lobot's machine)
- **Mode**: AppleScript (not Private API)
- **Access**: Via `bluebubbles` skill in StarforgeOS

## Limitations

- AppleScript mode is slower than Private API
- Some features (reactions, tapbacks) may not work
- Relies on Mac Mini being online
